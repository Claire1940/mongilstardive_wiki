## モンスターリング収集システム

『モンギル：スターダイブ』の核となるのが「モンスターリング」システムです。モンスターが単なる敵である従来のRPGとは異なり、本作では相棒の**ニャンナーズ**を使って倒した敵を吸収し、「モンスターリング」へと変化させます。これらは装備可能なアクセサリーとして機能し、操作キャラクターに大幅なステータスブーストと固有のパッシブ能力を付与します。

### モンスターリングの機能
各キャラクターは最終的に複数のモンスターリングを装備できるようになり、それらは可愛いアクセサリーとしてキャラクターモデル上に表示されます。これらのクリーチャーは、特定の戦闘条件で発動する「リンクチェーン」能力を提供します。

| モンスターリングの種類 | 主な恩恵 | 発動条件 |
| :--- | :--- | :--- |
| **濡れた葉のスライム** | 攻撃力 +20% ブースト | 攻撃をヒットさせた時 |
| **フォークマガー** | 攻撃力の大幅上昇 | 回避カウンター実行時 |
| **ゴーレム・ガーディアン** | 防御シールド | 低HP時にキャラクターを交代した時 |

### 合成と変異
エンドゲームの最も奥深い要素の一つに、2体のモンスターリングを組み合わせて「子」を作る合成があります。新しく生まれたクリーチャーは親の特性を継承したり、ギャンブル的に全く新しい高品質なステータスを獲得したりできます。また、33%の確率で**変異（Mutation）**が発生し、ユニークな外見と強化されたパッシブ能力を持つモンスターリングが誕生します。

## 戦闘メカニズムとチームシナジー

『モンギル：スターダイブ』の戦闘はテンポが速く、属性が重視されます。プレイヤーは3人のユニットでチームを編成し、敵の弱点を突きながら、モンスターリングとの「リンクチェーン」を維持するためにキャラクターを切り替えて戦います。

1.  **ロールの専門化**: キャラクターは**デストロイヤー**（スタッガー特化）や**アサシン**（バーストダメージ特化）などのクラスに分類されます。
2.  **属性相性**: 火、風、氷、土、雷の5つの主要属性があります。エリートボスは、強力な「アルティメット」攻撃を中断させるために、特定の属性ダメージを必要とすることが多いです。
3.  **スイッチスキル**: 敵の攻撃の瞬間にキャラクターを交代させると、敵をスタンさせることができ、大きなカウンターダメージのチャンスが生まれます。
4.  **オートクリア機能**: 素材集めのために、一度手動で攻略したドメインは「掃討」またはオートクリアが可能で、時間を大幅に節約できます。

## FAQ（よくある質問）

**Q: モバイルデバイスの現在のモンギルのゲーム容量はどれくらいですか？**

A: 2026年4月のリリース時点で、AndroidおよびiOSの合計**モンギルのゲーム容量**は約18〜22GBです。これには初期の全アセットとボイス付きストーリーコンテンツが含まれます。

**Q: 低スペックPCでも『モンギル：スターダイブ』をプレイできますか？**

A: 最適化はされていますが、安定した体験のためには少なくとも8GBのメモリと専用GPUが必要です。容量が足りない場合、PCクライアントではインストールする言語パックを選択して、全体のストレージ使用量を削減することができます。

**Q: ストレージが足りなくなった場合、ゲームサイズを小さくする方法はありますか？**

A: はい。設定メニューから「クリア済みストーリーアセット」や使用していないボイスパックを削除できます。メインキャンペーン終了後であれば、これにより2〜4GBのスペースを節約できることが多いです。

**Q: PCとモバイルの間でクロスセーブに対応していますか？**

A: はい、『モンギル：スターダイブ』はフルクロスプログレッションに対応しています。PCとモバイルデバイスで同じアカウントを使用できるため、外出先でもモンスターリングの管理が可能です。
