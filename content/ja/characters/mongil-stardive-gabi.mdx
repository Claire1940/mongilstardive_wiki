## ガビ：キャラクタープロフィールと基本ステータス

ガビは「ホタル森」に住む妖怪たちの勇敢なリーダーです。無鉄砲な外見とは裏腹に、人間界と妖怪界の微妙なバランスを保つ守護者としての役割を担っています。ゲームプレイにおいては、重い一撃と独自の「石化」状態異常を駆使して脅威を無力化する近接ファイターです。

純粋なバーストアタッカーとは異なり、ガビは戦闘時間が長引くほど強くなる「スロースターター」なプレイスタイルに焦点を当てています。そのため、長期戦となるエンカウントではT0.5（トップクラス）の選択肢となります。以下はレベル1時点の基本ステータスです。

## スキルキットとメカニクスの詳細

**モンギル：スターダイブのガビ**の核となるのは、「霊力」を生成・消費する能力です。このリソースは単なるダメージバフではなく、彼女の通常攻撃の機能を根本的に変化させ、標準的なアタッカーからクラウドコントロールのマシンへと変貌させます。

### パッシブ：トッケビの力
これがガビのキットの心臓部です。通常攻撃コンボの4段目をヒットさせると、霊力が1スタック付与されます（最大3スタック）。さらに、この攻撃は戦場に「岩」を生成する確率があります。この岩は単なる障害物ではありません。物理ダメージまたは地属性ダメージで破壊されると爆発し、周囲の敵に「石化」を付与し、ガビにさらに霊力スタックを与えます。

### 通常攻撃：トッケビバット
バットを激しく振り回し、物理ダメージを与えます。コンボの最後の一撃は地属性ダメージに変化します。
*   **強打 (Crushing Blow):** 霊力が3スタックある状態で攻撃ボタンを長押しすると、ガビは「強打」を繰り出します。この技は石化を付与し、莫大な地属性ダメージを与えます。
*   **無力化:** 敵の特殊攻撃に合わせて強打のタイミングを合わせると、敵を一時的に無力化し、その攻撃をキャンセルさせることができます。「強打者の意志」がアクティブな場合、即座に霊力スタックを獲得できます。

### 究極技：トッケビの領域
巨大な「木造の守護像」を召喚します。これは間違いなく、現在の征伐モードにおけるファイタークラスで最高の究極技の一つです。発動時に即座に地属性ダメージを与え、地の状態異常を付与します。像がアクティブな間：
1.  ガビの通常攻撃が強化され、地属性ダメージに変化します。
2.  霊力が定期的に付与されます。
3.  チームメイトに地属性ダメージアップと防御力アップのバフが付与されます。

## ガビの最適ビルド

**モンギル：スターダイブのガビ**のダメージ出力を最大化するには、地属性への変換と通常攻撃の頻度にシナジーがあるアーティファクトに注目する必要があります。究極技によって全てのダメージが地属性に変換されるため、「サージングエネルギー（エネルギー噴出）」効果が非常に優先されます。

| アーティファクト枠 | 推奨アイテム | 主な効果 |
| :--- | :--- | :--- |
| **メインスロット** | **トッケビが跪く者へ** | 地属性攻撃後、地属性ダメージが30-60%増加。 |
| **サブスロット** | **タッグロック** | 20ヒット後、攻撃力が5.5%増加。 |
| **第3スロット** | **岩拳の男** | スタッガー状態（ふらつき）のボスに対するクリティカルダメージが11%増加。 |
| **リンクチェーン** | **アヴァルダン** | ファイタークラスに必要なステータススケーリングを提供。 |

> **⚠️ 注意:** クリティカル率のステータスを無視しないでください。ガビの「強打」は倍率が高く、これらの攻撃でクリティカルが発生しないと、バーストフェーズ中のDPSが大幅に低下します。

## 超越とステージ進行 (A1-A6)

ガビを超越させることで、彼女は頼れるアタッカーから止められない破壊神へと進化します。各段階で、クールダウン管理や立ち上がりの遅さといった数少ない弱点を補う特定のユーティリティが提供されます。

| 段階 | 名称 | 効果の概要 |
| :--- | :--- | :--- |
| **A1** | もう少し強く叩く？ | 強打使用時に特殊スキルのクールダウンをリセット。 |
| **A2** | 大振りが来るぞ！ | スイッチスキルが石化を付与。通常攻撃4段目で必ず岩を生成。 |
| **A4** | 障害物を粉砕せよ！ | **ボスへのダメージが50%増加**。石化状態のターゲットへのダメージが100%増加。 |
| **A6** | 真の友の守護者 | 木造の守護像の効果と、究極技中のガビのダメージが2倍になる。 |

A4からA6への飛躍は、**モンギル：スターダイブのガビ**がオフェリアやサングンといった他の星5ファイターを、特にボス戦において圧倒し始めるポイントです。

## 最適なチーム編成

ガビは地属性のスペシャリストであるため、敵の地属性耐性を下げるキャラクターや、彼女の高い基本ステータスをさらに伸ばすバフを持つキャラクターと組ませることで真価を発揮します。

### 地属性支配チーム
*   **ガビ (メインDPS):** 霊力のスタックと強打の実行に集中します。
*   **フランシス (サポート):** スイッチスキルを通じて、ガビの攻撃力を爆発的に高める不可欠なバフを提供します。
*   **エリー (ブレーカー):** 敵のスタッガーゲージ（グロッキーゲージ）を素早く削るために使用します。敵がスタッガー状態になれば、ガビは中断されることなくフルコンボを叩き込めます。

### 戦略的なローテーション
1.  **エリー**で開始し、敵のグロッキーゲージを減少させます。
2.  **フランシス**に切り替えて、攻撃バフを付与します。
3.  **ガビ**をタグインし、スイッチスキル「フレー！」を使用して「強打者の意志」を獲得します。
4.  通常攻撃で岩を生成・破壊して石化を狙い、ボスがスタッガー状態になったら究極技を解放します。

## ゲームモード別の性能分析

2026年現在の『モンギル：スターダイブ』において、ガビはT0.5ユニットと評価されています。これは、絶対的な「必須」壊れユニットのわずか一歩手前でありながら、特定の役割においてはクラス最高であることを意味します。

*   **ワールド / 冒険:** 持続的なクラウドコントロールにより、エリートモンスターの群れを処理する際、オフェリアよりも安定感があります。
*   **征伐 (ボス戦):** ここが彼女の最も輝く場所です。彼女の究極技は、パーティ全体に地属性ダメージと防御力のバフを与えるため、ボス戦において最高峰の性能と広く見なされています。
*   **次元の裂け目:** 効果的ではありますが、単体攻撃に重点を置いていることと立ち上がりの時間が必要なため、大量の弱い敵を処理する速度はサングンやミナに一歩譲ります。

## FAQ

**Q: 初回バナーを逃した場合、どうすればガビを入手できますか？**

A: ガビは現在「新しい仲間スペシャルバナー」で確定入手ユニットとしてラインナップされています。「約束のコンパス」を使用して50回ガチャを引いた後、確定星5キャラクターとして彼女を選択できます。

**Q: 汎用性ではガビとサングンのどちらが優れていますか？**

A: コンテンツによります。時間制限のある次元の裂け目などではサングンの方が範囲殲滅速度が速い傾向にありますが、**モンギル：スターダイブのガビ**は石化と無力化メカニクスがあるため、ボス戦や高難易度の手動操作においては大幅に優れています。

**Q: ガビの装備で最も重視すべきステータスは何ですか？**

A: 攻撃力以外では、「ファイター熟練度」を優先すべきです。彼女の最適アーティファクトである「トッケビが跪く者へ」は、熟練度レベルに基づいて莫大なクリティカルダメージと攻撃力バフを提供するため、終盤のダメージを伸ばす最も効率的な方法となります。

**Q: 石化は全てのボスに効きますか？**

A: ほとんどのボスに石化は効きますが、一部のボスは高い耐性を持っており、複数回の付与が必要になる場合があります。ガビの岩を生成する能力は、石化をトリガーする機会を頻繁に作るため、この耐性を突破するのに役立ちます。
