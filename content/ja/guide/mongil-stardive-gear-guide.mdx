## 装備の基本：スロットとメインステータス

『モンギル：スターダイブ』では、各キャラクターに「頭部」「胴部」「手袋」「履物」の4つの主要な装備スロットがあります。サブステータスは獲得時や製作時にランダムに決定されますが、最初の3つのスロットのメインステータスは固定されています。履物は、独自の「熟練度」メカニズムがあるため、ビルドの成否を分ける重要なパーツとなります。

| 装備タイプ | 固定メインステータス | 重要性 |
| :--- | :--- | :--- |
| **頭部** | HP | 高難易度のコンクエストステージでの生存に不可欠。 |
| **胴部** | 防御力 | 被ダメージを軽減。デストロイヤーやタンクに必須。 |
| **手袋** | 攻撃力 | 全ロールにおける基礎ダメージの主要な源。 |
| **履物** | **役割熟練度** | **重要：** ロールがキャラクターと一致している場合のみ機能。 |

> **⚠️ 注意：** 履物の「役割熟練度」は必ず確認してください。エステルのような「アサシン」キャラクターに「ファイター熟練度」のブーツを装備させても、メインステータスの恩恵は一切受けられません。

## 最強の星5属性セット

高レベルの製作とコンクエストモードをアンロックしたら、星5の属性セットを目指すべきです。これらは、スペシャルスキル、クリティカル、属性状態異常などの特定の戦闘トリガーに連動した、強力な2セットおよび4セットボーナスを提供します。

### アビスセット（風）
アビスセットは現在、風属性アタッカーのゴールドスタンダードです。スペシャルスキル使用時に5秒間、風属性ダメージを10%アップし（2セット）、さらにクリティカル発生時に風属性ダメージを10%アップします（4セット）。これはエステル、特に彼女の「ダストクラウド」ゾーンと組み合わせるのに最適です。

### アービターセット（氷）
オフェリアのような氷属性ファイター向けに設計されたアービターセットは、交代スキル（Switch Skills）に焦点を当てています。キャラクターを入れ替えた際や、すでに氷状態異常に陥っている敵を攻撃した際に氷属性ダメージバフを付与します。頻繁にパーティを交代させる高強度のプレイスタイルに適しています。

### フンネルの咆哮（火）
火属性チームには「フンネルの咆哮」が最適です。敵の状態異常ステータスに基づいて、火属性ダメージとクリティカルダメージを強化します。火状態異常の付与と敵の防御力削りに長けたフレアに強く推奨されます。

## 主要キャラクターのおすすめアーティファクト

『モンギル：スターダイブ』のアーティファクトは、通常の防具とは異なる働きをします。アサシン、デストロイヤー、ファイター、サポーターの4つのクラスに分類されています。アーティファクトを選ぶ際は、星の数だけでなく「発動条件」を重視する必要があります。100%発動できる条件を持つ星4アーティファクトは、チームが満たせない条件を持つ星5アーティファクトよりも優れています。

| キャラクター | 役割 | 推奨アーティファクト | 主な効果 |
| :--- | :--- | :--- | :--- |
| **エステル** | アサシン | **風を宿したほうき** | 風属性アタッカーの攻撃力とクリダメを強化。 |
| **ギワン** | サポーター | **心の琴線で奏でるメロディ** | 交代スキル後、チーム全体のダメージを大幅強化。 |
| **オフェリア** | ファイター | **氷点下の裁定** | コンボチェーンとスキルダメージを強化。 |
| **フランシス** | サポーター | **あふれる愛** | 交代時にバフとクリティカルユーティリティを強化。 |

## 装備の製作と最適化の方法

製作システムは、エピソード1の序盤にあるサイドクエスト「自分でやってみよう！（Do It Yourself!）」でアンロックされます。中盤から終盤にかけて成功するためには、「つなぎ」の装備から最適化されたセットへと移行する必要があります。

1.  **素材を集める：** 毎日5回の「依頼」を活用して、特定のモンスターから素材を狙いましょう。星5素材については、**コンクエスト**のボスに挑戦する必要があります。
2.  **レベル3の法則：** 星5装備は最大4つのサブステータスを持ちます。レベル1で2つ、レベル3で3つ目が表示されます。最初の3つのサブステータスが悪い場合（例：DPS装備に固定値のHPなど）、強化を止めて他の装備の餌にしましょう。
3.  **履物を優先する：** 製作がアンロックされたらすぐに、メインDPSの役割に一致する履物の入手を目指してください。
4.  **採掘：** ルビーやサファイアなどの希少な鉱石は、マップの隅に隠れていることが多いです。目的地に急ぐだけでなく、周囲を探索しましょう。

> **💡 ヒント：** 強化済みの装備を新しい装備の素材として使用すると、経験値の約70%が還元されます。序盤の難所を突破するために、星4装備をためらわずにレベルアップさせましょう。

## よくある装備の失敗例

2026年のサービス開始初期、多くのプレイヤーが苦戦するのは、本作を単なるステータスチェックのゲームとして扱ってしまうからです。成功の鍵は「モンスターリンク」と「交代（スワップ）」メカニズムの理解にあります。

*   **控えキャラの価値を無視する：** ギワンやフランシスのようなキャラクターは、操作していない時でも価値を発揮します。彼らに『心の琴線で奏でるメロディ』のようなアーティファクトを装備させることで、DPSに交代するたびに強力なダメージ倍率を乗せることができます。
*   **序盤からセット効果にこだわりすぎる：** 超序盤では、セット効果を揃えるために弱い部位を装備するよりも、攻撃力のメインステータスが高い単体装備の方が強力です。まずは全スロットを埋めることを優先し、その後にセットを厳選しましょう。
*   **耐性減少を軽視する：** 『山頂の威厳』のような星4セットは耐性減少を提供します。星5装備を持っていたとしても、メインアタッカーのダメージを伸ばすために、サポーターにあえて敵の耐性を下げる星4セットを着せることが有効な場合があります。

## FAQ

**Q: アビスセット用の星5モンスター素材はどこで入手できますか？**

A: これらの素材は主に**コンクエスト**モードで入手できます。より高いステージのボスを倒すと、ボスのソウルを特定の星5素材や完成済みの装備品と交換できるようになります。

**Q: このガイドではエンドゲームでも星4装備を推奨していますか？**

A: 星5装備が最終目標ですが、『ムーンシャドウ』（クリティカル補助）や『山頂の威厳』（耐性減少）などの特定の星4セットは、中盤以降もサポーター用として十分に通用します。

**Q: 新しいブーツを装備したのにダメージが増えないのはなぜですか？**

A: ブーツの**役割熟練度**を確認してください。その役割（ファイター、アサシン、デストロイヤー、サポーター）がキャラクターの役割と一致していない場合、履物のメインステータスは無効化されます。

**Q: 星5装備にはサブステータスがいくつ付きますか？**

A: 星5装備は最大4つのサブステータスを持ちます。レベル1で2つが表示され、レベル3、6、9、12、15、18で追加のスロット解放または強化が行われます。
