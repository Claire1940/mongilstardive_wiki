## 装備の基本：種類とステータス

「モンギル：スターダイブ」のすべてのキャラクターは、頭防具、胴防具、手袋、靴の4種類の装備を装着できます。装備の種類とメインステータスの関係を理解することが、最適化への第一歩です。ほとんどのスロットはメインステータスが固定されていますが、靴スロットは可変の熟練度ボーナスがあるため、後半のビルドにおいて最も重要になります。

### 装備スロットとメインステータス

| 装備の種類 | 固定メインステータス | 重要性 |
| :--- | :--- | :--- |
| **頭防具** | HP（体力） | バーストダメージに対する生存力 |
| **胴防具** | 防御力 | 持続的な物理/属性ダメージの軽減 |
| **手袋** | 攻撃力 | 素のDPSスケーリングの主要ソース |
| **靴** | ロール熟練度 | 特定クラスの有効性を倍増させる |

序盤の優先事項は、とにかくすべてのスロットを埋めることです。チーム全員が少なくとも3つ星の装備を揃えるまでは、セット効果を気にする必要はありません。4つ星（紫）や5つ星（金）の装備を製作し始めたら、キャラクターのロールに合った適切な装備を組み合わせることに焦点を移しましょう。

## 製作で力を手に入れる

装備がランダムドロップでしか手に入らない他の多くのガチャRPGとは異なり、「モンギル：スターダイブ」は堅牢な製作システムを重視しています。この機能は、エピソード1のサイドクエスト「DIY（ドゥ・イット・ユアセルフ）！」を通じて早い段階で解放されます。製作では特定のセットを狙うことができるため、ランダムドロップによるフラストレーションを軽減できます。

### 製作プロセス
1.  **メニューにアクセス：** メイン画面から「製作」をタップします。
2.  **セットを選択：** 一般的な装備、または「アビス」や「アービター」などの特定のハイティアセットから選択します。
3.  **素材を集める：** オープンワールドで採掘した鉱石と、依頼や討伐で入手したモンスター素材の組み合わせが必要になります。
4.  **製作実行：** ゴールドと素材を消費してアイテムを生成します。メインステータスは固定ですが（靴を除く）、サブステータスはランダムに決定されることに注意してください。

> **💡 ヒント：** マップ上でルビーやサファイアなどの希少な鉱石を見かけたら、必ず採掘しておきましょう。これらは隅の方に隠れていることが多く、5つ星装備を製作する際の主なボトルネックとなります。

## セット効果の理解

セット効果は、あらゆるメタビルドの根幹です。同じシリーズの装備を2ピースまたは4ピース装着すると、属性ダメージやユーティリティを大幅に強化する独自のパッシブ効果が発動します。2026年現在、プライムスターセットは対人・高難易度プレイにおけるゴールドスタンダードとなっています。

### 注目の装備セット

| セット名 | 2ピース効果 | 4ピース効果 | 推奨ロール |
| :--- | :--- | :--- | :--- |
| **アビス** | 特殊スキル使用時、5秒間風属性ダメージ+10% | クリティカルヒット時、5秒間風属性ダメージ+10% | 風属性DPS（エステルなど） |
| **アービター** | スイッチスキル使用時、5秒間氷属性ダメージ+10% | 氷状態異常の敵に対し、氷属性ダメージ+10% | 氷属性DPS / サポーター |

これらのハイティアセットを製作するには、特殊な素材が必要です。例えば、アビスセットには**ヴォイドの欠片**、**ムーンストーン**、**エメラルド**が必要です。これらは、ボスが段階的に強くなる代わりに報酬も良くなる「討伐」モードで集めるのが最適です。

## 靴とロール熟練度の最適化

多くのプレイヤーが犯す共通のミスは、靴の「ロール熟練度」ステータスを無視することです。頭防具や手袋とは異なり、靴のメインステータスはキャラクターの特定のロールと一致している場合にのみ恩恵をもたらします。

*   **ファイター熟練度：** 集団戦に長けたファイターのステータスを強化します。
*   **デストロイヤー熟練度：** フレアのようなキャラクターが敵のよろめきゲージをより速く溜めるために不可欠です。
*   **アサシン熟練度：** 行動不能（CC）デバフを利用するキャラクターのバーストダメージを最大化します。

> **⚠️ 警告：** 現在のメインDPSとロールが一致しないからといって、靴をすぐに捨てないでください。サブメンバーや将来の引きのために保管しておきましょう。ロールが一致した5つ星の靴は希少であり、決して単なる強化素材として使ってはいけません。

## サブステータスの厳選戦略

目的のセットを製作したら、この**モンギル：スターダイブ 装備ガイド**の次のステップは、完璧なサブステータスを狙う「厳選」です。各装備には最大4つのサブステータスが付与されます。製作直後には2つのサブステータスが見えており、アイテムがレベル3に達すると3つ目が追加されます。

### サブステータス強化の節目
| レベル | 効果 |
| :--- | :--- |
| **Lv.1** | 2つのサブステータスが表示 |
| **Lv.3** | 新しいサブステータスが1つ追加 |
| **Lv.6** | 既存のランダムなサブステータスが強化 |
| **Lv.9** | 最後のサブステータスが追加 |
| **Lv.12-18** | さらにランダムな強化 |

最も効率的な最適化方法は、レベル3時点での最初の3つのサブステータスが良くない場合に強化を止めることです。望ましいサブステータスは一般的に、**会心率**、**会心ダメージ**、**攻撃力**です。DPSアイテムに複数の「耐性」ステータスが付いた場合は、そのアイテムを新しい製作の経験値素材として使うのが得策です。投資した経験値の約70%を回収できます。

## 素材を効率的に集める

高い製作レベルを維持するには、日課の管理が重要です。依頼は1日5回までに制限されています。これらは、次に作りたい装備に必要なモンスター素材に基づいて慎重に選ぶべきです。

1.  **デイリー依頼：** 4つ星セットの特定の構成パーツをドロップするモンスターをターゲットにします。
2.  **討伐モード：** 5つ星素材の主要な入手先です。ステージ10のボスを倒すと、ボストークンをそのボスのシグネチャー装備や素材と直接交換できるようになります。
3.  **アクティブなフレンド：** 毎日のスタミナを受け取るためにアクティブなフレンドを追加しましょう。スタミナが多いほど、採掘ランやモンスター素材集めの回数が増えます。
4.  **オズのパズル：** マップに隠されたこれらの目的を達成して、最大スタミナ容量を永続的に拡張するスクロールを入手しましょう。

## ロール別の装備フォーカス

3人チームにおけるキャラクターの役割に応じて、装備戦略を変える必要があります。

*   **ファイター：** 攻撃力とHPのバランスを重視します。ファイターは敵に囲まれているときにより多くのダメージを与えるため、攻撃と同じくらい生存も重要です。
*   **アサシン：** 純粋な攻撃ステータス。会心率と会心ダメージのサブステータスを探しましょう。アサシンは敵が打ち上げられたり、よろめいたりした時に輝くため、装備の発動タイミングをこれらの窓に合わせるのが鍵です。
*   **デストロイヤー：** 靴のデストロイヤー熟練度を最優先します。彼らの仕事は、アサシンが仕留めに入りやすいようにボスのよろめきゲージをできるだけ早く削ることです。
*   **サポーター：** クールタイム短縮（セット効果で利用可能な場合）とHP/防御力に焦点を当て、回復やバフを提供し続けられるように場に留まらせます。

## よくある質問（FAQ）

**Q：いつからセット効果を意識し始めるべきですか？**

A：4つ星（紫）装備を安定して製作できるようになったら、セット効果に注目し始めるのがおすすめです。序盤は、無理に低ティアのセット効果を狙うよりも、手持ちの装備で純粋なステータスを上げる方が有益です。

**Q：既存の装備のサブステータスを変更できますか？**

A：現在、個別のサブステータスをリロール（再抽選）することはできません。別のサブステータスが必要な場合は、新しく製作するかドロップを狙う必要があります。そのため、リソースを節約するためにレベル3での「チェック」が不可欠です。

**Q：製作のためのゴールドを稼ぐ最善の方法は何ですか？**

A：ゴールドはデイリーミッションと次元の裂け目のクリアで稼ぐのが一番です。後半の製作コスト（1箇所16,000ゴールド以上）はすぐに膨れ上がるため、低ティアの3つ星装備を過剰に強化するのは避けましょう。

**Q：このガイドにおいて属性耐性は重要ですか？**

A：一般的には攻撃ステータスが好まれますが、特定の属性耐性（火、氷、風、雷）は、強力な属性状態異常を与えてくる後半の討伐ボスに対して非常に有効です。ニッチな戦闘用に、耐性の高い装備を数箇所取っておくと良いでしょう。
