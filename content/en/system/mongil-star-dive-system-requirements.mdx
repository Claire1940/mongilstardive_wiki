This MONGIL STAR DIVE system page summarizes practical hardware expectations and optimization options for stable frame pacing.

## Recommended checks

- Keep drivers and OS updated.
- Leave enough free storage for patches.
- Lower shadow and post-processing settings first.
- Enable battery or thermal performance mode on mobile when needed.

## Common performance fixes

1. Restart after major patch downloads.
2. Clear cached assets if stuttering appears.
3. Disable background downloads and overlays.
4. Switch to a closer region server when latency spikes.

Use these steps before deeper troubleshooting.
