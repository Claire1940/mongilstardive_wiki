> **💡 Tip:** If you are struggling with frame rates, try lowering the "Effects Quality" first. This setting controls the flashy combat animations which are the primary cause of GPU spikes.

## FAQ

**Q: Can I run Mongil: Star Dive on a laptop with integrated graphics?**

A: It is possible if you have a modern processor (like an Intel Core i5 12th Gen or newer with Iris Xe graphics), but you will likely need to play at 720p resolution on the lowest settings to maintain 30 FPS.

**Q: Why does the game require 50GB of storage if the download is only 7GB?**

A: The 7GB is the compressed initial installer. Once the game is installed, it downloads high-resolution textures, voice-over files for multiple languages, and expansive world data, which expands the total size significantly.

**Q: Is Windows 11 mandatory for the mongil star dive pc requirements?**

A: No, Windows 10 is still supported for the minimum requirements. However, Windows 11 is recommended for better compatibility with DirectX 12 and improved memory management.

**Q: Does the game support controllers on PC?**

A: Yes, the PC version has native support for Xbox and PlayStation controllers, which is often the preferred way to play for high-action combat.
