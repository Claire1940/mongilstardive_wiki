## Optimizing Mongil Star Dive Linux Performance

Running the game is only half the battle; ensuring it runs at a stable 60 FPS requires some fine-tuning. Because Mongil Star Dive is a visually vibrant game, shader compilation can cause stuttering during the first few minutes of gameplay.

### Performance Tweaks

*   **Enable Gamemode:** In Heroic settings, ensure "Enable Feral GameMode" is toggled on. This gives the game higher CPU priority.
*   **Use DXVK:** Ensure DXVK is enabled (it usually is by default in Proton) to translate DirectX 11/12 calls to Vulkan.
*   **Login Issues:** If the Google Login option fails to open your browser, try signing in with a standard email and password. Some Linux sandbox environments prevent the launcher from communicating with external web browsers.

> **💡 Tip:** If the game fails to launch, check if you have the `vulkan-loader` and 32-bit Vulkan drivers installed on your host system.
