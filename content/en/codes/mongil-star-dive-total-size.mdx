## Optimizing Storage and Performance

If you are struggling with the **mongil star dive total size** on a mobile device, there are several steps you can take to optimize the experience without sacrificing too much visual fidelity.

*   **Delete Language Packs:** The game features high-quality voice acting. If you only play with one language, delete the unused voice packs in the settings menu to save up to 2 GB.
*   **Clear Cache:** Periodically clearing the in-game cache can remove old patch fragments that are no longer needed.
*   **Texture Scaling:** On PC, using DLSS or FSR can allow you to download lower-resolution base textures while maintaining a sharp image.
