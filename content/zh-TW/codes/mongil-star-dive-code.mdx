## 如何兌換序號

目前有兩種主要方法可以領取《魔物學園：STAR DIVE》中的獎勵。根據您的平台（PC 或行動裝置），其中一種方法可能會更方便。

### 方法 1：遊戲內兌換 (Android/PC)
對於大多數玩家來說，遊戲內選單是輸入 **mongil star dive code** 最快的方式。

1. 啟動遊戲並點擊右下角的**設定**圖示（齒輪符號）。
2. 導覽至左側側邊欄的**其他**分頁。
3. 找到**兌換碼**按鈕並點擊。
4. 在文字框中輸入序號並按下**確認**。
5. 檢查您的遊戲內信箱以領取道具。

### 方法 2：網頁門戶兌換 (iOS/全平台)
由於 iOS 裝置通常會限制遊戲內優惠券選單，因此官方網頁門戶是最佳替代方案。此方法需要您的**會員編碼**（玩家 ID）。

| 步驟 | 動作 | 詳細資訊 |
| :--- | :--- | :--- |
| **1** | 找到玩家 ID | 前往設定 > 其他 > 複製會員編碼。 |
| **2** | 訪問門戶網站 | 導覽至 Netmarble 官方優惠券頁面。 |
| **3** | 輸入資料 | 輸入您的會員編碼和序號。 |

## 支持者獎勵與獨特序號

除了在社群媒體上分享的公開序號外，《魔物學園：STAR DIVE》還擁有一套獨特的「支持者」系統。這允許玩家支持他們喜愛的內容創作者，以換取專屬的一次性兌換碼。這些代碼非常有價值，因為它們通常包含標準公開序號中沒有的星光水晶等高級貨幣。

要參與支持者計畫：
1. 訪問透過遊戲社群論壇連結的官方支持者網頁。
2. 選擇一位要支持的創作者（這通常是免費的）。
3. 幾小時後，檢查「我的頁面」下的「領取福利」部分。
4. 複製您獨特的一次性序號，並使用上述方法進行兌換。

> **警告：** 獨特的支持者序號一旦生成即與特定帳號綁定。分享這些序號將導致它們對您自己失效。

## FAQ

**問：為什麼我的 mongil star dive code 無效？**

答：有三個常見原因：序號已過期、輸入時大小寫錯誤，或者您已在帳號上兌換過。檢查是否出現「Exchange Error」訊息以確認是否已被使用。

**問：在哪裡可以找到我的會員編碼？**

答：前往遊戲內設定選單，選擇「其他」分頁，並在底部查看您的玩家 ID 或會員編碼。旁邊通常有一個「複製」按鈕以便操作。

**問：支持者序號是免費的嗎？**

答：是的，在 Netmarble 生態系統中支持創作者通常是一項免費操作，在短暫的處理期後會獎勵您獨特的序號。

**問：序號多久發布一次？**

答：新序號通常在開發者直播、重大遊戲更新或遊戲在應用商店達到特定的下載里程碑時發布。請每週回來查看最新更新。
