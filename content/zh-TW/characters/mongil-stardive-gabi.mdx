## 嘉比：角色簡介與基礎屬性

嘉比是螢火蟲之森中妖怪們大膽的領袖。儘管外表魯莽，但她是維護人類與妖怪境界之間微妙平衡的守護者。在遊戲玩法上，她是一位近戰戰士，依靠重擊和獨特的「石化」狀態效果來中和威脅。

與純粹的爆發型刺客不同，嘉比專注於慢熱成長型的戰鬥風格。她的效能在戰鬥進行中不斷提升，使其成為長線戰鬥中 T0.5 等級的選擇。以下是她在 1 級時的基礎屬性：

| 屬性類別 | 基礎值 (Lv. 1) | 成長潛力 |
| :--- | :--- | :--- |
| **HP (生命值)** | 792 | 高 |
| **ATK (攻擊力)** | 106 | 中 |
| **DEF (防禦力)** | 38 | 高 |
| **Groggy DMG (虛弱傷害)** | 15 | 卓越 |
| **Brawler Stat (格鬥數值)** | 58 | 標準 |

> **💡 提示：** 雖然嘉比是一名戰士，但她高額的 HP 和 DEF 成長使她能夠在缺乏專職守護者（Guardian）的隊伍中充當副坦克。

## 技能組與機制拆解

**mongil stardive gabi** 的核心在於她產生與消耗「靈力」的能力。這種資源不僅僅是傷害加成；它從根本上改變了她普通攻擊的運作方式，使她從標準的格鬥者轉變為控場機器。

### 被動：小鬼之力 (Dokkaebi Power)
這是嘉比技能組的引擎。普通攻擊連段的第 4 擊命中時會獲得一層靈力（最高 3 層）。此外，這一擊有機率在戰場上生成一顆「岩石」。這些岩石不僅是障礙物；當被物理或大地傷害破壞時，它們會爆炸，對附近敵人造成石化效果，並為嘉比提供另一層靈力。

### 普通攻擊：小鬼球棒 (Dokkaebi Bat)
嘉比狂野地揮舞球棒，造成物理傷害。連段的最後一擊會轉換為大地傷害。
*   **碎裂擊 (Crushing Blow)：** 在擁有 3 層靈力時長按攻擊鍵，嘉比會釋放碎裂擊。此招式會造成石化效果並輸出巨額大地傷害。
*   **中和 (Neutralization)：** 如果您在敵人釋放特殊攻擊時精準施放碎裂擊，可以暫時中和他們，有效地取消其動作；若「強棒意志」處於激活狀態，還能立即獲得靈力層數。

### 終極技：小鬼領域 (Dokkaebi's Domain)
嘉比召喚巨大的守護木像。這可以說是目前征服模式中戰士類別最強的終極技。它會造成即時的大地傷害並施加大地異常狀態。當木像存在時：
1.  嘉比的普通攻擊獲得強化並轉為大地傷害。
2.  週期性獲得靈力。
3.  隊友獲得大地傷害與防禦力加成。

## Mongil Stardive 嘉比最佳出裝

為了最大化 **mongil stardive gabi** 的傷害輸出，您必須專注於能與她的大地元素轉換和普攻頻率產生協同作用的神器。由於她的終極技會將所有傷害轉換為大地傷害，「能量湧動」效果是優先考量的重點。

| 神器欄位 | 推薦裝備 | 核心效果 |
| :--- | :--- | :--- |
| **主神器** | **令小鬼下跪之人** | 大地攻擊後提升大地傷害 30-60%。 |
| **副欄位** | **追蹤鎖 (Taglock)** | 達成 20 次命中後提升攻擊力 5.5%。 |

| **連結鏈** | **阿瓦爾丹 (Avardan)** | 為戰士職業提供必要的屬性縮放。 |

> **⚠️ 警告：** 不要忽視暴擊率屬性。嘉比的「碎裂擊」擁有極高的倍率，如果這些攻擊未能暴擊，會顯著降低她在爆發階段的 DPS。

## 覺醒與階段進程 (A1-A6)

覺醒嘉比能讓她從可靠的格鬥者蛻變為勢不可擋的力量。每個階段都提供了特定的效用，解決了她少數的弱點，如冷卻時間管理和暖機時間。

| 階段 | 名稱 | 效果摘要 |
| :--- | :--- | :--- |
| **A1** | 再打重一點？ | 施放碎裂擊時重置特殊技能冷卻時間。 |
| **A2** | 大揮擊預備！ | 切換技能會施加石化；第 4 擊必定生成岩石。 |
| **A4** | 摧毀所有障礙！ | **對首領傷害提升 50%**；對石化目標造成 100% 額外傷害。 |
| **A6** | 真摯友誼的守護者 | 終極技期間守護木像效果與嘉比傷害翻倍。 |

從 A4 到 A6 的飛躍是 **mongil stardive gabi** 真正開始超越歐菲莉亞（Ophelia）或山軍（Sangun）等其他 5 星戰士的地方，尤其是在首領密集的內容中。

## 最佳團隊陣容

嘉比是大地屬性專家，這意味著當她與能夠削減大地抗性或提供與她高基礎屬性掛鉤的增益的角色搭配時，表現最為出色。

### 大地支配隊
*   **嘉比 (主 DPS)：** 專注於堆疊靈力並執行碎裂擊。
*   **弗朗西斯 (輔助)：** 弗朗西斯透過切換技能提供關鍵增益，成倍提升嘉比的攻擊力。
*   **艾莉 (破防位)：** 艾莉用於快速削減敵人的虛弱條（Stagger Gauge）。一旦敵人進入虛弱狀態，嘉比即可安全地執行完整連段而不受干擾。

### 戰鬥循環策略
1.  以 **艾莉** 開場，削減敵人的虛弱條。
2.  切換至 **弗朗西斯** 施加攻擊性增益。
3.  換入 **嘉比**，使用她的切換技能「呼哈！」獲得強棒意志。
4.  執行普通攻擊以生成岩石，破壞它們觸發石化，然後在首領虛弱時釋放終極技能。
