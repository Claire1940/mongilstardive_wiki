## 쿠폰 사용 방법

현재 '몬길: 스타 다이브'에서 보상을 받는 방법은 크게 두 가지입니다. 사용 중인 플랫폼(PC 또는 모바일)에 따라 더 편리한 방법을 선택할 수 있습니다.

### 방법 1: 게임 내 입력 (안드로이드/PC)
대부분의 플레이어에게 게임 내 메뉴를 통한 입력이 **몬길 스타 다이브 쿠폰**을 사용하는 가장 빠른 방법입니다.

1. 게임을 실행하고 오른쪽 하단 모서리에 있는 **설정** 아이콘(톱니바퀴 모양)을 누릅니다.
2. 왼쪽 사이드바에서 **기타** 탭으로 이동합니다.
3. **쿠폰 입력** 버튼을 찾아 누릅니다.
4. 텍스트 상자에 코드를 입력하고 **확인**을 누릅니다.
5. 게임 내 우편함을 확인하여 아이템을 수령합니다.

### 방법 2: 웹 포털 입력 (iOS/모든 플랫폼)
iOS 기기는 게임 내 쿠폰 메뉴가 제한되는 경우가 많으므로, 공식 웹 포털을 사용하는 것이 가장 좋습니다. 이 방법을 사용하려면 **멤버 코드**(플레이어 ID)가 필요합니다.

| 단계 | 동작 | 상세 내용 |

| **1** | 플레이어 ID 찾기 | 설정 > 기타 > 멤버 코드 복사로 이동합니다. |
| **2** | 포털 방문 | 공식 넷마블 쿠폰 페이지로 이동합니다. |

서포터 프로그램에 참여하려면:
1. 게임 커뮤니티 포럼을 통해 연결된 공식 서포터 웹페이지를 방문합니다.

3. 몇 시간 후 "마이 페이지"의 "혜택 받기" 섹션을 확인합니다.
4. 고유한 일회용 코드를 복사하여 위의 방법으로 사용합니다.

> **주의:** 고유 서포터 코드는 생성 시 특정 계정에 귀속됩니다. 이 코드를 공유하면 본인의 계정에서 사용할 수 없게 됩니다.

### 중복 캐릭터 (돌파)
레이나와 같은 캐릭터의 중복 복사본을 얻는 것은 실제로 매우 유익합니다. 중복 유닛을 통해 "각성" 단계를 해제할 수 있으며, 이는 기본 스탯을 크게 높이고 패시브 능력을 강화합니다. 각성이 완료된 4성 캐릭터는 게임 초중반에 기본 5성 캐릭터보다 더 뛰어난 성능을 발휘하기도 합니다.
