## 크로스 프로그레션 및 동기화 활성화 방법

**몬길 스타 다이브 크로스플레이**를 최대한 활용하려면 특정 계정 연결 절차를 따라야 합니다. 모바일에서 시작했다면 PC에서 별도의 계정을 생성하지 말고 동일한 자격 증명을 사용하여 로그인하세요.

1.  **설정 열기:** 메인 메뉴의 계정 탭으로 이동합니다.
2.  **계정 연결:** 넷마블 ID, 구글 또는 애플 ID 중에서 선택합니다.
3.  **이메일 인증:** 데이터를 보호하기 위해 인증 프로세스를 완료합니다.
4.  **PC 로그인:** PC 클라이언트를 실행하고 초기 부팅 시 "기존 계정" 옵션을 선택합니다.

### 몬스터링 동기화
**몬길 스타 다이브 크로스플레이**를 사용하면 독특한 "나너스(Nanners)"와 같은 몬스터링 컬렉션이 일정하게 유지됩니다. 이 크리처들은 전투력의 핵심입니다. 예를 들어, 나너스와 같은 몬스터링은 다른 차원과 연결된 특수한 소화관을 가지고 있어 침으로 공격적인 몬스터를 제압할 수 있습니다. 이 데이터는 서버 측에 저장되므로 레벨과 "몬스터링" 상태(완전히 길들여진 야수임을 나타냄)는 휴대폰과 데스크톱 모두에서 동일하게 유지됩니다.

**Q: 몬길 스타 다이브 크로스플레이를 플레이스테이션과 엑스박스 간에 이용할 수 있나요?**

A: 2026년 초 현재, **몬길 스타 다이브 크로스플레이**의 주요 초점은 PC와 모바일(iOS/Android) 간의 연결입니다. 콘솔 버전은 넷마블에서 논의된 바 있으나 현재 별도의 개발 주기에 있습니다.

**Q: PC와 모바일에서 플레이하려면 게임을 두 번 구매해야 하나요?**

A: 아니요, 이 게임은 공유 계정 시스템을 갖춘 부분 유료화(Free-to-Play) 모델을 따릅니다. 지원되는 모든 플랫폼에서 클라이언트를 다운로드하고 기존 자격 증명으로 로그인하여 여정을 계속할 수 있습니다.

**Q: 모바일 사용자와만 플레이하고 싶은 경우 크로스플레이를 비활성화할 수 있나요?**

A: 현재 이 게임은 빠른 매치메이킹과 활발한 세계를 보장하기 위해 통합 서버 아키텍처를 사용합니다. 모든 플레이어가 동일한 버전의 하나니스에 공존하므로 크로스플레이를 비활성화하는 옵션은 없습니다.

**Q: 유료 재화(다이아몬드/골드)가 플랫폼 간에 동기화되나요?**

A: 골드 및 퀘스트 포인트와 같이 획득한 재화는 완벽하게 동기화됩니다. 그러나 플랫폼 정책에 따라 iOS에서 구매한 일부 "유료 다이아몬드"는 iOS 기기에서만 보이고 사용할 수 있을 수 있습니다. 단, 해당 재화로 구매한 아이템은 모든 플랫폼에서 사용할 수 있습니다.
